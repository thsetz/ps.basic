Basic package
=============

The ps.basic package holds 4 Modules

   - Basic (the Basic class and some helper functions)
   - fsm   (a skelleton to imlement finite state machines (with
     autogenerated pictures of the fsm))
   - Patterns (some Patterns for shell commands)
   - ps_shell_log ( a cmd_line tool to create logging message
     useable from a shell)
